
# FoodyApp

FoodyApp is an application where you can filter and list recipes and save your favorite recipes.

# Used Libraries
- Coroutines
- Dagger Hilt
- Retrofit
- Room
- Navigation Components
- Coroutines Flow
- ViewBinding
